# Privacy Policy

GPT Input Render is a browser extension that renders Markdown and common LaTeX syntax in ChatGPT user messages.

## Data Processing

The extension reads visible message content on supported ChatGPT pages only to render it locally in the browser. This processing happens on the user's device.

## Data Collection

GPT Input Render does not collect, store, sell, transfer, or share user data. It does not send message content, browsing history, identifiers, credentials, or analytics to the developer or to any third party.

## Remote Code

The extension does not load or execute remote JavaScript or WebAssembly. All runtime code is included in the extension package.

## Permissions

The extension runs only on the ChatGPT page URLs declared in its manifest. It does not request additional browser permissions.

## Contact

For support, open an issue at https://github.com/GaBoron/gpt-input-render/issues.
